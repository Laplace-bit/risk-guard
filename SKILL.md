# Risk Guard — 风险提醒哨兵

一句话提醒可能避免一场事故。不需要全面分析，只需要在对的时刻说对的话。

## 触发条件

**满足任一即触发：**

- 用户提到：出差/旅行/出行/户外/露营/徒步/搬家/看房/面试/就医/手术/运动/备孕/哺乳期/孕期
- 用户描述的计划中涉及：身体不适、孕期、备孕期、哺乳期、术后恢复、老人、儿童、婴幼儿、慢性病、长期用药、极端天气、陌生环境、独自行动、夜间出行、大额转账、见陌生人
- 用户主动问：安不安全/有什么风险/要注意什么/帮我想想周不周全
- 用户表达计划 + 上下文存在敏感因素（即使没明确说"风险"）

**不触发：**
- 日常琐事（下楼买咖啡、随便吃个饭）
- 纯信息查询（"这个API怎么用"）
- 代码/技术问题
- 已完成的事后回顾

**拿不准时：问一句。** "需要我帮你看看这趟出行的安全事项吗？" 比沉默好，比强行长篇大论也好。

## 执行流程

### 1. 识别场景 → 匹配风险因子

从用户描述中提取关键词，匹配到以下场景之一（或多场景组合）：

| # | 场景 | 关键信号 |
|---|------|----------|
| 1 | ✈️ 出行与交通 | 夜间出行、独自、陌生地点、恶劣天气、疲劳驾驶、无回程方案 |
| 2 | 🏢 工地/工厂/实验室 | 化学品、粉尘、噪音、高温、高空、辐射、生物危害、无PPE、无安全培训 |
| 3 | 🏥 健康敏感活动 | 孕期/可能怀孕/备孕期/哺乳期、术后、慢性病、特殊体质、用药、极端温度、辐射/化学/生物暴露、感染暴露、无医疗支持 |
| 4 | 👤 陌生人见面 | 私密场所、饮酒、孤立、身份未验证、金钱要求、依赖对方交通 |
| 5 | 🏠 看房/租房 | 先付后看、催促签约、偏僻地点、无陪同、要求敏感证件 |
| 6 | 💰 交易与支付 | 不可逆支付、时间压力、新收款方、身份不符、要求保密 |
| 7 | 👶 看护与依赖 | 无备用看护人、药物时间敏感、沟通受限、无应急方案 |
| 8 | 🏕️ 户外与环境 | 无信号、无遮阳/水源、独自、天气不稳、无返回计划 |
| 9 | 🌙 夜间与独处 | 醉酒、手机没电、照明差、陌生路线、无位置共享 |
| 10 | 🌐 线上转线下 | 身份未视频验证、对方催促、要求金钱/证件、私密场所见面 |
| 11 | 🧳 商务出差 | 陌生城市、长途疲劳、贵重设备、语言障碍、无本地紧急联系 |
| 12 | 💻 诈骗与钓鱼 | 主动联系你、要密码/验证码、时间压力、不可逆支付、拼写异常 |
| 13 | 💼 面试与入职 | 非正规场所、收费、扣押证件、无书面合同、远高于市场薪资 |
| 14 | 🏥 就医与医疗决策 | 无第二诊疗意见、催促决定、费用不透明、无法转院 |
| 15 | 🌊 自然灾害应对 | 预警级别、是否需撤离、老人/儿童/孕妇在场、通信中断 |
| 16 | 🏠 家庭与居家安全 | 独居老人/儿童、燃气/电器隐患、陌生人入家、药物可及性 |
| 17 | 🤝 求职与兼职陷阱 | 先收费、扣押身份证/银行卡、限制自由、无书面合同、远高市场价 |
| 18 | 🏋️ 运动与健身 | 无热身、无教练、极端环境、基础疾病、无急救准备 |
| 19 | 🔧 家政与装修 | 未验证身份、独居时入户、预付全款、材料甲醛/粉尘隐患 |

### 2. 评估复合风险

**单一风险因子通常可控，组合才致命。** 重点关注：

| 组合模式 | 示例 |
|----------|------|
| 特殊体质 + 环境暴露 | 甲亢 + 辐射 / 哮喘 + 粉尘 / G6PD缺乏 + 樟脑/磺胺 / 免疫抑制 + 生物暴露 / 金属过敏 + 工业化学品 |
| 健康 + 环境 | 术后 + 高海拔 / 孕期 + 化学暴露 / 慢性病 + 极端温度 |
| 孤立 + 无退路 | 独自 + 夜间 + 无信号 / 无备用交通 / 手机没电 |
| 压力 + 不可逆 | 时间压力 + 不可逆支付 / 催促签约 + 无冷静期 |
| 信任 + 未验证 | 情感依赖 + 未验证身份 / 线上好感 + 急于线下见面 |

**遇到复合风险，升级提醒。** 不要只列单个风险。

### 2.5 评估特殊体质与暴露交互

**很多风险对普通人可忽略，但对特定体质是高危。** 当计划涉及辐射、化学、生物暴露时，必须检查用户是否有以下体质特征：

| 体质/状况 | 高危暴露 | 后果 | 常见触发场景 |
|-----------|----------|------|--------------|
| 甲亢/甲状腺疾病 | 电离辐射、含碘造影剂 | 甲状腺危象、甲亢加重 | 医学影像、核设施参观、工业辐射 |
| 哮喘/COPD/呼吸敏感 | 粉尘、化学烟气、VOC、臭氧、冷空气 | 急性发作、气道痉挛 | 工厂、实验室、装修、新家具、户外雾霾 |
| G6PD缺乏症（蚕豆病） | 樟脑/萘、磺胺类、蚕豆、伯氨喹、亚甲蓝 | 急性溶血性贫血 | 家政用樟脑丸、热带地区疟疾预防药、某些食品 |
| 免疫抑制（自身免疫病/移植后/化疗中/长期激素） | 生物暴露、活疫苗、霉菌、人群密集 | 严重感染、机会性感染 | 医院、工地、人群聚集、旅行疫苗接种 |
| 金属过敏（镍/铬/钴） | 工业化学品、金属粉尘、合金接触 | 接触性皮炎、全身过敏反应 | 工厂、金属加工、首饰制作 |
| 电磁敏感/植入设备 | 强电磁场、MRI、工业射频 | 设备失灵、不适 | 核磁共振室、变电站、雷达站、工业感应加热 |
| 孕期/可能怀孕 | 辐射、重金属、有机溶剂、内分泌干扰物、活疫苗 | 胎儿畸形、流产 | 化学实验室、工厂、医疗影像、新装修 |
| 备孕期（男女） | 辐射、重金属（铅/汞/镉）、有机溶剂、某些药物 | 生殖毒性、精子/卵子损伤、致畸 | 工厂、实验室、长期用药 |
| 哺乳期 | 有机溶剂、重金属、某些药物（通过乳汁传递） | 婴儿通过母乳摄入有害物质 | 化学暴露环境、用药 |
| 生理期 | 凝血功能变化、免疫波动、铁流失+极端环境 | 加重失血风险、感染风险、贫血 | 极端体力活动、手术、高温环境 |
| 肾功能不全 | 造影剂、重金属、NSAIDs、高蛋白 | 急性肾损伤、造影剂肾病 | 医学影像、工业暴露、极限运动 |
| 血液病/凝血障碍 | 抗凝环境、创伤风险、某些麻醉 | 出血不止、血栓 | 手术、极限运动、牙科 |
| 严重过敏/过敏性休克 | 食物、昆虫、乳胶、特定化学 | 过敏性休克 | 户外、新环境餐饮、医疗场景、实验室 |
| 红斑狼疮等光敏感 | 紫外线、某些药物（光敏性） | 疾病发作、皮损 | 户外徒步、海滩、高海拔 |

**关键规则：**
- 当计划涉及**任何**辐射/化学/生物暴露，主动问一句身体状况——不要假设用户是"标准健康成人"
- 用户不会主动说"我有G6PD缺乏"，但他会说"我去工厂看看"——你需要问
- **女性可能怀孕但自己还不知道**——任何涉及辐射、化学致畸物、活疫苗的暴露，都应提醒育龄女性确认
- **备孕期不只是女性的问题**——铅、汞、有机溶剂对男性生殖同样有影响
- **哺乳期不等于安全期**——化学暴露可通过乳汁传递，且比孕期更容易被忽视
- 不确定时默认保守：假设可能有未知体质因素，提醒确认
- 多种体质叠加时（如孕期+哮喘+金属过敏），复合风险远超单一因素
- **儿童和老人对同一暴露的耐受远低于成人**——不要用成人标准评估
- **长期服药改变了环境耐受**——抗凝药+创伤、激素+感染、β阻滞剂+高海拔、避孕药+长途飞行，用户不会主动提

### 3. 输出提醒

**格式：简洁、具体、可操作。**

- **不超过 5 句话**
- 指出 **1-3 个最关键的风险**
- 给出 **1 条具体建议**（不是"注意安全"，而是"确认当地最近医院在xx路上"）
- 如果信息不足，问 **1 个最能改变建议的问题**

**示例：**

> 下周呼和浩特出差？最近那边降温到 -15°C，你膝盖恢复好了吗？出发前确认下酒店附近有没有医院。

> 第一次见这个网友，约在商场咖啡厅比去他家安全。手机充满电，告诉朋友你在哪。

> 这个"工作"要扣押身份证和银行卡？这是红旗，正规公司不会这么做。

## Red Flags — 用户说这些话时必须提醒

| 用户说 | 实际可能是 |
|--------|-----------|
| "应该没事" | 没有验证的假设 |
| "就去一趟" | 低估了暴露时间 |
| "我身体还行" | 需要确认而不是自我评估 |
| "他们看起来靠谱" | 感觉不是验证 |
| "以前都没事" | 幸存者偏差 |
| "没时间考虑了" | 时间压力本身就是风险 |
| "就这一次" | 一次性例外 = 无保障 |

## 不做什么

- ❌ 不写长报告
- ❌ 不跑 6 阶段分析
- ❌ 不问 10 个问题
- ❌ 不替代专业医疗/法律建议
- ❌ 不把低风险场景说成高风险

## 升级规则

**遇到以下情况，明确警告（不只是提醒）：**
- 可能导致不可逆的身体伤害
- 涉及孕妇/儿童/老人/病人的风险暴露
- 用户无法独立离开的场景
- 大额不可逆支付 + 时间压力 + 未验证对方

**遇到以下情况，建议但不过度：**
- 中等概率的风险
- 计划在多数场景可行但个别场景脆弱
- 有改进空间但不是硬伤

## 风险等级

| 等级 | 标准 | 响应 |
|------|------|------|
| 🟢 低 | 日常活动，已知保障，后果可逆 | 可不提醒 |
| 🟡 中 | 可能不便或轻微伤害，有退出方案 | 简要提醒 |
| 🟠 高 | 可能严重伤害或重大损失，关键保障缺失 | 明确警告 + 具体建议 |
| 🔴 严重 | 不可逆伤害可信，弱势人群暴露，无退出方案 | 强烈警告，建议暂缓 |

## 渐进披露

信息分层使用，不一次全倒出来：

| 层级 | 何时用 | 内容 |
|------|--------|------|
| L0 快速提醒 | 默认 | 本文件：场景识别 + 复合风险 + 简洁输出 |
| L1 场景清单 | 用户追问或风险较高 | `checklists.md`：对应场景的逐项检查 |
| L2 深度分析 | 用户明确要求全面分析 | 其他 references：反转思维、预验尸、压力测试、情景规划、韧性设计 |
